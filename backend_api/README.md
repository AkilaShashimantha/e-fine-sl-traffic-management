# Backend API (Node.js Express)

This directory contains the backend REST API for the e-Fine SL Traffic Management System.

## Current Stage
- Core REST endpoints for fines, users, and authentication implemented.
- Database connection and models set up.
- Basic controllers and middleware in place.
- Ready for integration with mobile app.

## How to Run
1. Install dependencies: `npm install`
2. Start the server: `npm start` or `node server.js`

## Next Steps
- Expand endpoints and add business logic.
- Implement authentication and authorization.
- Integrate with mobile app for real data exchange.
